# Vue project
Small site where you can upload images. VueJs handling front and node Express server for back.

# Server

## Project setup
```
npm install
```
## Run
```
node .\index.js
```
Server responds at http://localhost:4000/

# Client

## Navigate
```
cd .\client\
```
## Project setup
```
npm install
```
## Run
```
npm run serve
```
Client responds at http://localhost:8080/
